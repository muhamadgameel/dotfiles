# NeoVim

## Dependecies

- [git](https://git-scm.com/)
- [bat](https://github.com/sharkdp/bat) (preview) required by telescope
- [fd](https://github.com/sharkdp/fd) (finder) required but telescope
- [ripgrep](https://github.com/BurntSushi/ripgrep) (finder) required by telescope

### Mac OS

- [ninja](https://github.com/ninja-build/ninja/wiki/Pre-built-Ninja-packages) build system (required by 'sumneko_lua' language server)

## Formatters
- [prettier](https://prettier.io/) (npm install -g prettier)
- [LuaFormatter](https://github.com/Koihik/LuaFormatter) (luarocks install --server=https://luarocks.org/dev luaformatter)
- [rustfmt](https://github.com/rust-lang/rustfmt) (rustup component add rustfmt) (comes when installing rust using rustup)
